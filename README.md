## Compile on IRIX (by flexion)

To compile on IRIX you need SDL2 and SDL2_mixer. 
SDL2 is used for audio playback and the optional windowed mode -w

make              # dynamic (default)
make STATIC=1     # static SDL2 + SDL2_mixer
